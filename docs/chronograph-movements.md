# Chronograph Movement Reference (DRAFT — needs your verification)

The core of GemHunter's edge: knowing a movement's worth from the listing even when
the seller doesn't name it. Two halves:
1. **Movement traits** — column-wheel (premium) vs cam (fine, common)? Era? Grade?
   Can the user service it?
2. **Reference → caliber map** — a vague listing ("Wakmann triple calendar") still
   resolves to a caliber ("Valjoux 730") → value signal.

> ⚠️ **My draft — please red-pen.** Confidence column = where to focus (H/M/L).
> We treat **Ranfft** as the source of truth for verification.
> CW = column-wheel (VALUE↑). Cam is still bought/serviced, just lower premium —
> **except** the 7750 family, where *grade/finish* drives value (see "Grade matters").

Scope = **Swiss** vintage/neo-vintage chronographs. Russian / Chinese / Japanese
movements are **excluded** (see bottom). **Age floor ~1960** — older watches are too
fragile to service (see playbook §2d: shock protection, stud carrier, hairspring).
The pre-'60s CW greats below are **admired but age-capped** — surface only if exceptional.

## Grade matters (same base caliber, different value)

The **7750** especially — value tracks **brand + finish**, not the base:
- **Base 7750** → beater, "always works," low premium.
- **Chronometer-grade** (e.g. **Breitling** Navitimer/Chronomat) → higher, accurate.
- **In-house-finished** (**IWC** cal. 79xx: own escape wheel, optimized pallet-fork angle, own mainspring/gears, gold vs. chrome-plated; ~1980s–early-2000s pre-Richemont) → **prized.**

→ Scoring should carry a **grade modifier** keyed on brand, not just the caliber.

## Column-wheel-vs-cam quick tells (reading a photo)

- **Era + brand**: "too early to be cam" → CW. *(Need per-brand cutoff years — biggest open item.)*
- **Sub-dial layout**: Venus 175 = **45-min** counter (right) vs the typical **30-min** counter at 3 o'clock on most cam movements.
- **Movement shot present** → seller likely knows it's worth something; **absence** is common on underpriced gems.
- _To add: register count/positions, hand styles, exotic/special dials._

## Excluded — never surface

- **Russian** (Poljot, Vostok, etc.)
- **Chinese** movements / clones
- **Japanese** (Miyota, Citizen, **Seiko incl. chronographs** — even 6138/6139). Only possible exceptions: vintage **Credor / Grand Seiko / King Seiko**.
- **Valjoux 7733 / 7734 / 7736** and other 7733 variants — **personal dislike** (cam-actuated, noisy, not enjoyable to service). Skip even though serviceable & parts-available.

## Data sources & how to cobble this together

**No public source classifies column-wheel vs cam** — it's editorial, not a DB field.
That flag is *our* curated layer (the edge). Public sources only supply technical
caliber data to enrich around it:

| Source | Size | Quality | Access | Use for us |
|--------|------|---------|--------|-----------|
| [Ranfft.org](https://ranfft.org/) | ~13k | auto-parsed | free site, **bot-blocks scraping (403)**; old ranfft.de dumps circulate | **source of truth** (user trusts it) — verify per-caliber |
| [17jewels.info](https://17jewels.info/) | ~1k | hand-curated | free site | accuracy cross-check |
| [watch-movements-archive.com](https://watch-movements-archive.com/) | ~2k | community, structured | free site | secondary |
| [The Watchmakers App](https://thewatchmakersapp.com/) | ~11k | structured | **has API** (paywalled) | only if we scale past chronographs |

**Plan:** scope to this finite chrono set (not 13k movements); seed CW/cam + grade from
curation (the part no source gives us); enrich tech fields per-caliber from Ranfft as
needed (targeted pulls, not wholesale scraping); grow the reference→caliber map from the
user's buys + live listings.
